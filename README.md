# stock-prediction
数据挖掘期末大作业。

使用了许多股票里的指标。用了**QDA**、**RandomForest**、**GBDT**等多个模型进行训练和预测，并在最后进行模型融合。

实现了对**上证指数**的预测。
|  模型   | 2020年12月准确度  | 2021年1月初准确度 |
|  ----  | ----  | ----|
| QDA  | 73% | 100% |
| 融合模型  | 65% | 75% |

未来准确度如何不敢保证。:joy:
